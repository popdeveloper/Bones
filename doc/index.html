Bones - skeletal animation for jPCT

Bones is a skeletal animation API for jPCT. It's aimed to be small library with no additional dependencies where possible. Minimum Java version is 1.5.

It can load Collada skins via Ardor3D's Collada loader and Ogre3D skins via jME's OgreXml loader. After this initial loading, skins can be saved in a compact binary form. Saved skins can be later reloaded with loader, with no dependencies to Ardor3D or jME. Either way after loading they are ready in jPCT terms. They have mesh data, textures coordinates and skinning information. It's enough to set their textures and call build() to prepare them to be added into a jPCT world.

Bones initially started as a partial/modified port of Ardor3D's animation system to jPCT. Then it evolved to match jPCT semantics and to load Ogre3D skins.

Bones can be downloaded from here. Javadocs can be found here. It contains the source code, necessary libraries, import scripts, Eclipse project files and a few sample applications. Bones is definitely free software with a "Do whatever with it" license. However, please note, adapted sources and sample model data may still be subject to provider's license terms.

As Bones is at a early phase, expect changes in the API. Even for later phases, the author claims no warranty to maintain backward compatibility.

For discussion and questions, visit the Bones thread at jPCT forums
